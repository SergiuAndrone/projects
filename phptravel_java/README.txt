- compile using: javac -cp ".:lib/okio-1.14.0.jar:lib/okhttp-3.11.0.jar:lib/guava-25.0-jre.jar:lib/commons-exec-1.3.jar:lib/byte-buddy-1.8.15.jar:lib/client-combined-3.141.59.jar:lib/client-combined-3.141.59-sources.jar:lib/selenium-remote-client-2.0a5.jar:" Tests.java
- run using: java -cp ".:lib/okio-1.14.0.jar:lib/okhttp-3.11.0.jar:lib/guava-25.0-jre.jar:lib/commons-exec-1.3.jar:lib/byte-buddy-1.8.15.jar:lib/client-combined-3.141.59.jar:lib/client-combined-3.141.59-sources.jar:lib/selenium-remote-client-2.0a5.jar:" Tests

Running the script on windows:

- Download the chromedriver for windows
- compile and run using the same commands as on macos but REPLACE : with ;
